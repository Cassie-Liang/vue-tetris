# vue-tetris

通过开发一款俄罗斯方块的小游戏，来了解webpack和vue的使用。

## 开发环境

1. 下载源代码
2. 安装依赖库，`npm i`
3. 安装Webpack，`npm i -g webpack`
4. 启动构建工具，`webpack --watch`

## 在线DEMO

https://ideatry.github.io/vue-tetris/build/index.html

## 运行游戏

直接用浏览器打开 /build/index.html 即可

## 游戏控制

*键盘操作*

左移：方向键`左`
右移：方向键`右`
变形：方向键`上`
加速：方向键`下`

*按钮操作*

左移：按钮`左`
右移：按钮`右`
变形：按钮`上`
加速：按钮`下`

*手势操作*

左移：左划
右移：右划
变形：上划
加速：下划
